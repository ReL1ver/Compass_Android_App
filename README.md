# JustCompass

Android приложение цифрового компаса, использующее сенсоры устройства.
## Описание
Приложение отображает направление на магнитный север с помощью акселерометра и магнитометра устройства.
Компас отображает:
- текущий азимут (угол);
- стрелку направления;
- 8 сторон света;
- вибрацию при достижении севера.

Также используется автоматическая калибровка магнитного поля.
## Функции
- цифровой компас;
- плавная анимация стрелки;
- 8 сторон света (С, СВ, В, ЮВ, Ю, ЮЗ, З, СЗ);
- вибрация при достижении севера;
- сглаживание данных сенсоров.
## Используемые технологии
- Kotlin;
- Jetpack Compose;
- Android Sensors API.
## Требования
- Android 10+;
- устройство с магнитометром и акселерометром;
## Установка и запуск
1. Скачать проект.
2. Открыть в Android Studio.
3. Собрать проект.
4. Установить APK на устройство.
